Proyecto React

Este proyecto se basa en una SPA (Single Page Aplication) simulando la estructura y funcionalidad de un e-commerce. Entre sus funciones se encuentran: Muestra dinámica del contenido en una sola página, sistema de carrito personalizable, paginación, sistema de filtrado de productos y un sistema de checkout que devuelve los datos de compra y un ticket. Todo esto funcionando de manera asincrónica e interactuando con una base de datos en Firebase, actualizando el stock de productos en base a la ventas y el almacenamiento de los datos de compra de los usuarios.

Proyecto Backend

Este proyecto esta pensado para implementar el funcionamiento backend básico para un e-commerce, centrandose mas en las funcionalidades que los estilos. El mismo está desarrollado utilizando Node.js y Express para el backend, y MongoDB como base de datos para almacenar información sobre productos y carritos de compras. El mismo funciona de manera local pero se puede observar el codigo libremente en el repositorio del proyecto donde se incluyen las colecciones de la base de datos.
